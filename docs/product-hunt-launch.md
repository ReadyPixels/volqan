# Product Hunt Launch Assets

## Soft Launch (v0.5.0-beta)

**Tagline:** The open-source CMS, admin panel, and page builder — all in one Docker command.

**Description (240 chars):**
Volqan is a self-hosted application framework that generates your admin panel, REST + GraphQL API, and page builder automatically from your content schema. No vendor lock-in. Ships free forever.

**First comment (Maker comment):**
> Hey Product Hunt! 👋
>
> We built Volqan because every project we started needed the same three things: a CMS, an admin panel, and a backend API — and we kept rebuilding them from scratch.
>
> Volqan solves this by generating all three from a single content schema. You define your data model once, and Volqan produces the admin UI, the REST API, the GraphQL API, and the content management interface automatically.
>
> **What makes it different:**
> - Fully self-hosted — one `docker compose up` and you own everything
> - Extension engine — install community extensions from Bazarix marketplace
> - Theme engine — swap the entire admin UI design without touching code
> - Embedded AI assistant — connects to OpenAI, Claude, Gemini, or local Ollama
> - Open Core Attribution License — free forever, with optional Pro tier for white-label
>
> We'd love your feedback on what features matter most for v1.0. What would make you switch from your current CMS stack?
