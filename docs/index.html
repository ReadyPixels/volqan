Volqan

Build Anything. Own Everything.

The open-source application framework that is a CMS, an admin panel, a visual page builder, and a backend application platform — all in one. Ship free. Stay free. Own every byte.

Get Started GitHub

$ npx create-volqan-app my-project

$ cd my-project && pnpm dev

Why Volqan?

Most frameworks make you choose. You either get a CMS or a backend framework or a page builder — never all three, never without a vendor lock-in. Volqan removes that choice.


You define your data. Volqan generates the admin panel, the REST API, the GraphQL API, the forms, and the content management interface — automatically. Then you ship it with a single Docker command and walk away.

Core Features

Everything you need to build powerful applications without the complexity.

⚙️

Two Usage Modes

Developer Mode for code-first control or Visual Mode for point-and-click building. One engine powers both — switch anytime.

🔗

Auto-Generated APIs

Every content model instantly becomes a full REST API and GraphQL API. No routes to write, no resolvers to maintain.

🧩

Extension Engine

First-class extension system. Add admin menu items, custom pages, dashboard widgets, API routes, and more — all through TypeScript.

🎨

Theme Engine

Full design token system. Swap themes instantly. Every color, font, spacing, and animation is configurable.

🤖

Embedded AI Assistant

AI lives inside your admin panel. Connect to OpenAI, Claude, Gemini, or run locally with Ollama.

✨

Beautiful by Default

Built with shadcn/ui and Tailwind CSS v4. Premium SaaS look and feel out of the box.

🐳

Self-Hosted Forever

Docker-first. One command starts everything. No usage limits, no per-seat pricing, no vendor lock-in.
